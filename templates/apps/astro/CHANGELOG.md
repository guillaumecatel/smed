## [v0.0.0] - 2026-01-22

### Added

- Initial release: Astro app template for monorepos.
- Integrated Tailwind CSS, Paraglide (i18n), Playform compression.
- Node standalone server output with CSP enabled.
- Strict TypeScript config with path aliases and Astro types.
- E2E tests with Playwright.
- Integration tests with Vitest.
- Example components, layouts, middleware, and i18n setup.
- Clean script for all build/test artifacts.
- Dynamic OpenGraph image endpoint with Satori.
- SEO-friendly base layout.
- SEO best practices: robots.txt, sitemap.xml.
- PWA-ready: i18n webmanifest.
