## [v0.0.0] - 2026-01-25

### Added

- Initial release: Storybook + React app template for monorepos.
- Strict TypeScript configuration with JSX/TSX support.
- Preconfigured Storybook with example stories and custom decorators.
- E2E (Playwright) and integration (Vitest) test setup.
- Example React component (`Button`) and associated story.
- Structure ready for internationalization (i18n).
- Scripts for development, build, test, and clean.
